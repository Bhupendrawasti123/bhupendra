How This Website Works

This birthday celebration website shows motivational messages with a fun scoring system!

Features:

  • Customizable name and photo
  • Messages loaded from JSON
  • Multiple color themes
  • Confetti celebration
  • Sound effects (can be muted)
  • Scoring system with streaks

Scoring Rules:

  • Base Points: 10 points per message
  • Streak Bonus: 2x multiplier per consecutive click (max 5x)
  • Max Streak: 20 bonus points at 5+ streak
